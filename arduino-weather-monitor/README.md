# Arduino Weather Monitoring System

## Overview
A real-time temperature and humidity monitoring system using a DHT22 sensor. The system displays live environmental data on a 16x2 LCD and provides LED and buzzer alerts based on defined thresholds.

## Features
- Real-time temperature and humidity monitoring
- Multi-screen LCD display
- LED status indicators (Green, Yellow, Red)
- Buzzer alert system
- Max/Min temperature tracking
- Environmental condition classification (Cold, Comfortable, Warm, Hot, Dry, Humid)

## Hardware Components
- Arduino UNO
- DHT22 sensor
- 16x2 LCD display
- LEDs
- Buzzer
